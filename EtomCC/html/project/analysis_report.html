四.行业发展现状及趋势

1.行业市场规模及生命周期
a.市场容量、总体（国内外）市场规模
b.细分市场份额
c.行业发展速度、市场增长率、需求增长率
d.行业平均利润水平
e.行业生命周期分析
2.行业市场及竞争情况（参考波特五力模型中的行业内竞争强度）
a.供需关系（需求稳定性等
b.竞争者数量
c.行业内企业分类（战略或竞争策略小组、企业梯队）
d.行业内企业固定资产投资情况
e.行业内主要企业及其市场份额
f.行业内企业力量对比
g.行业市场集中度（国内外）
h.行业内总体竞争强度（激烈程度）、竞争方式（价格、质量、服务、品牌）、主要竞争策略
3.行业进入壁垒
a.业内已存在公司的优势（品牌、专利技术、资质、资金、人才、规模经济、政策、特许、自然垄断、资源供应等）
b.退出成本（固定资产投资、有形及无形资产专用性等）
c.业内公司对于进入者的反应
4.上游供应链分析（供应商讨价议价能力
a.上游行业状况，包括：供应商规模分布、供应商市场集中度，供应商竞争激烈程度
b.上游产品的标准化程度
c.上游产品的可替代性及重要性
d.供应商对本行业的依赖度分析（供应商成功需要的资源、利润来源是否与本行业关系密切，供应商的替代性，本行业是否是唯一的采购者）
e.供应商的纵向一体化倾向
f.上游供应产品的交易方式
g.本行业对供应商的依赖程度（行业需要的技能、资源、利润来源是否与供应商关系密切）
5.下游客户分析（客户的讨价议价能力）
a.下游行业状况，包括：客户规模分布、客户市场集中度，客户竞争激烈程度
b.本行业产品的标准化程度
c.本行业产品的可替代性及重要性
d.客户对本行业的依赖度分析（客户成功需要的资源、利润来源是否与本行业关系密切，客户的替代性，本行业是否是唯一的供应者）
e.客户的纵向一体化倾向
f.本行业产品的交易方式
g.本行业对下游客户的依赖程度（行业需要的技能、资源、利润来源是否与下游客户关系密切）
6.替代品的可替代性情况
a.行业产品替代品的种类、涉及行业
b.替代品替代行业产品的规模、增长率
c.替代品与行业产品的优劣比较、替代性分析（技术上和成本上的可替代性）
d.替代产品技术的发展情况等等
7.行业产业链的配套情况，比如供应链主要区域、产业链配套情况等（可以上面分析维度重复补充）
8.行业所处宏观经济环境（可参考PEST或PESTEL分析框架展开），包括政治、经济、社会、技术、环境、法律等各个外部环境情况，对于行业和产业、企业发展能够产生冲击的各个要素情况
9.行业发展的影响因素（有利及不利因素）
a.行业对资源和技术的要求（资金密集型、劳动密集型、技术密集型）
b.行业技术发展趋势、技术进步前景
c.国家宏观政策对于行业的影响（鼓励、限制、无影响） （可重复）
d.行业技术发展趋势，国际技术走向、发展前景分析、国际技术领先的国家、公司的名称、简介、技术领先之处、国内技术水平、发展趋势、与国外的技术差距、国内技术领先公司的名称、简介、技术领先之处
e.其他可能存在的制约因素（资源约束、环保要求、其他要求）
f.行业发展当前面临的主要问题及对策
